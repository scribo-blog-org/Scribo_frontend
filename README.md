# Scribo

Simple blog site project

## Technologies
- HTMl
- CSS, SCSS
- ReactJS
- JsonWebToken
- Dotenv

## Functionality

- Creating posts
- Reading posts
- Login/Register accounts
- Following/Unfollowing

## Install and run

```bash
git clone https://github.com/MaksimKosyanchuk/Scribo_frontend
cd news-site
```
set server link into `REACT_APP_API_URL` variable in .env file

```bash
npm install
npm start
```
